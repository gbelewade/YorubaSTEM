The greatest common divisor (GCD) of two integers is the largest positive integer that divides both without a remainder.	Pínpín àjọṣe púpọ̀jùlọ (GCD) àwọn òòkà odidi méjì jẹ́ òòkà odidi dáradára púpọ̀jùlọ tí ó pín méjèèjì láìsí ìyókù.
The least common multiple (LCM) of two integers is the smallest positive integer that is divisible by both.	Pínpín àjọṣe kéréjùlọ (LCM) àwọn òòkà odidi méjì jẹ́ òòkà odidi dáradára kéréjùlọ tí ó lè jẹ́ pínpín nípa méjèèjì.
A prime number is a natural number greater than 1 that cannot be formed by multiplying two smaller natural numbers.	Òòkà àkọ́kọ́ jẹ́ òòkà ṣáà tí ó tóbi ju 1 lọ tí a kò lè ṣẹ̀dá nípa ìsọdipúpọ̀ àwọn òòkà ṣáà méjì tí ó kéré.
The prime factorization of a number is the product of its prime factors.	Ìsọdipúpọ̀ àkọ́kọ́ òòkà kan jẹ́ ìsọdipúpọ̀ àwọn àkọkọ ìpín rẹ̀.
A function is continuous at a point if the limit of the function as the variable approaches that point exists and equals the function value at that point.	Ìfà jẹ́ alátìṣu ní ojú kan bí òdíwọ̀n ìfà náà bí onídìmọ̀ ṣe ńṣúnmọ́ ojú náà bá wà tí ó sì dọ́gba pẹ̀lú iye ìfà ní ojú náà.
A function is differentiable at a point if its derivative exists at that point.	Ìfà jẹ́ aláyàtọ̀sọ́tọ̀ ní ojú kan bí dérífátífù rẹ̀ bá wà ní ojú náà.
A limit is the value that a function approaches as the input approaches some value.	Òdíwọ̀n jẹ́ iye tí ìfà kan ṣúnmọ́ bí ìwọlé ṣe ńṣúnmọ́ iye kan.
An asymptote is a line that a curve approaches but never touches.	Àfọsẹ̀ jẹ́ ìlà tí kọ́fù kan ṣúnmọ́ ṣùgbọ́n kò lè fọwọ́kan.
A local maximum is a point where the function value is greater than at nearby points.	Àgbàjùlọ abẹ́lé jẹ́ ojú níbi tí iye ìfà tóbi ju ti àwọn ojú tímọ́.
A local minimum is a point where the function value is less than at nearby points.	Àkéréjùlọ abẹ́lé jẹ́ ojú níbi tí iye ìfà kéré ju ti àwọn ojú tímọ́.
A global maximum is the highest point on the entire function.	Àgbàjùlọ akáríayé jẹ́ ojú gíga jùlọ lórí gbogbo ìfà náà.
A global minimum is the lowest point on the entire function.	Àkéréjùlọ akáríayé jẹ́ ojú kéré jùlọ lórí gbogbo ìfà náà.
An even function satisfies f(-x) = f(x) for all x in its domain.	Ìfà onírúurú mú f(-x) = f(x) ṣẹ fún gbogbo x nínú àdúgbò rẹ̀.
An odd function satisfies f(-x) = -f(x) for all x in its domain.	Ìfà òrìtú mú f(-x) = -f(x) ṣẹ fún gbogbo x nínú àdúgbò rẹ̀.
A complex number has the form a + bi, where a and b are real numbers and i is the imaginary unit.	Òòkà àdàpọ̀ ní ọ̀nà a + bi, níbi tí a àti b jẹ́ àwọn òòkà gidi àti i jẹ́ ẹ̀yà àfojúró.
The absolute value of a number is its distance from zero on the number line.	Iye ajukájú òòkà kan jẹ́ ìjìnnà rẹ̀ láti òdo lórí ìlà-òòkà.
The factorial of a non-negative integer n, denoted n!, is the product of all positive integers less than or equal to n.	Ìṣesí òòkà odidi aláìdọ́wọ́lé n, tí a fì àmì hàn n!, jẹ́ ìsọdipúpọ̀ gbogbo àwọn òòkà odidi dáradára tí ó kéré ju tàbí dọ́gba pẹ̀lú n.
A permutation is an arrangement of a set of objects in a specific order.	Àtòsípo jẹ́ ìtò ìtò àwọn nkan ní ṣíṣe pàtó.
A combination is a selection of objects from a set without regard to order.	Àdàpọ̀ jẹ́ àṣàyàn àwọn nkan láti ìtò kan láìsí ìfẹ̀sẹ̀ sí ṣíṣe.
A sequence is an ordered list of numbers.	Ìtẹ̀lé jẹ́ àkọjọ àwọn òòkà tí a tò.
An arithmetic sequence has a common difference between consecutive terms.	Ìtẹ̀lé ìṣirò ní ìyàtọ̀ àjọṣe láàrín àwọn ẹ̀yà tó tẹ̀lé ara wọn.
A geometric sequence has a common ratio between consecutive terms.	Ìtẹ̀lé àjé ní ìbùpín àjọṣe láàrín àwọn ẹ̀yà tó tẹ̀lé ara wọn.
A recursively defined sequence has each term defined in terms of one or more previous terms.	Ìtẹ̀lé aṣàfihàn atẹyìnpadà ní ẹ̀yà kọ̀ọ̀kan tí a ṣàlàyé ní ọ̀nà ẹ̀yà kan tàbí jù bẹ́ẹ̀ lọ tẹ́lẹ̀.
A Fibonacci sequence is defined by F₀ = 0, F₁ = 1, and Fₙ = Fₙ₋₁ + Fₙ₋₂ for n > 1.	Ìtẹ̀lé Fíbónáṣì ni a ṣàlàyé bíi F₀ = 0, F₁ = 1, àti Fₙ = Fₙ₋₁ + Fₙ₋₂ fún n > 1.
The binomial theorem gives the expansion of (a + b)ⁿ.	Òfin ìdálẹ́jì fún wa ní ìgbóróṣẹ (a + b)ⁿ.
A set is a collection of distinct objects.	Ìtò jẹ́ àkójọ àwọn nkan ọ̀tọ̀ọ̀tọ̀.
The union of two sets A and B, denoted A ∪ B, contains all elements that are in A or B or both.	Àpapọ̀ àwọn ìtò méjì A àti B, tí a fì àmì hàn A ∪ B, ní gbogbo àwọn èròjà tí ó wà nínú A tàbí B tàbí méjèèjì.
The intersection of two sets A and B, denoted A ∩ B, contains all elements that are in both A and B.	Ìpàdé àwọn ìtò méjì A àti B, tí a fì àmì hàn A ∩ B, ní gbogbo àwọn èròjà tí ó wà nínú A àti B.
The complement of a set A, denoted A', contains all elements in the universal set that are not in A.	Àkúnlé ìtò A, tí a fì àmì hàn A', ní gbogbo àwọn èròjà nínú ìtò àkáríayé tí kò sí nínú A.
The empty set, denoted ∅, contains no elements.	Ìtò òfìfo, tí a fì àmì hàn ∅, kò ní èròjà kankan.
A Venn diagram is a graphical representation of sets.	Àwòrán Vẹ́nù jẹ́ àfihàn àwọrán àwọn ìtò.
A relation is a set of ordered pairs.	Ìbátan jẹ́ ìtò àwọn ìwọ̀n tí a tò.
The domain of a relation is the set of all first components of its ordered pairs.	Àdúgbò ìbátan jẹ́ ìtò gbogbo àwọn èròjà àkọ́kọ́ àwọn ìwọ̀n tí a tò rẹ̀.
The range of a relation is the set of all second components of its ordered pairs.	Àyè-kàn ìbátan jẹ́ ìtò gbogbo àwọn èròjà kejì àwọn ìwọ̀n tí a tò rẹ̀.
A function is a relation where each element in the domain corresponds to exactly one element in the range.	Ìfà jẹ́ ìbátan níbi tí èròjà kọ̀ọ̀kan nínú àdúgbò bá ẹ̀yà kan pàtó nínú àyè-kàn mu.
A linear equation in two variables has the form ax + by = c.	Ọ̀mì-dogbá nínú àwọn onídìmọ̀ méjì ní ọ̀nà ax + by = c.
A system of linear equations is a set of two or more linear equations.	Ètò àwọn ọ̀mì-dogbá jẹ́ ìtò àwọn ọ̀mì-dogbá méjì tàbí jù bẹ́ẹ̀ lọ.
The solution to a system of linear equations is the set of values that satisfies all equations in the system.	Ojútùú sí ètò àwọn ọ̀mì-dogbá jẹ́ ìtò àwọn iye tí ó mú gbogbo àwọn ọ̀mì nínú ètò náà ṣẹ.
Gaussian elimination is a method for solving systems of linear equations.	Ìmúkúrò Gáúsiánù jẹ́ ọ̀nà fún yíyanǹjú àwọn ètò ọ̀mì-dogbá.
A conic section is a curve obtained as the intersection of a plane and a double-napped cone.	Abala kọ́nìkì jẹ́ kọ́fù tí a rí gẹ́gẹ́ bí ìpàdé pẹpẹ kan àti abala-méjì kọ́nù.
A circle is a conic section formed when a plane intersects a cone perpendicular to the axis of the cone.	Ẹ̀ká jẹ́ abala kọ́nìkì tí a ṣẹ̀dá nígbà tí pẹpẹ bá da kọ́nù lọ́nà ọgbà sí ọ̀pọ̀n kọ́nù náà.
An ellipse is a conic section formed when a plane intersects a cone at an angle to the axis of the cone.	Ẹ̀láípsì jẹ́ abala kọ́nìkì tí a ṣẹ̀dá nígbà tí pẹpẹ bá da kọ́nù ní igun sí ọ̀pọ̀n kọ́nù náà.
A parabola is a conic section formed when a plane intersects a cone parallel to a line on the cone.	Kọ́nú-fífà jẹ́ abala kọ́nìkì tí a ṣẹ̀dá nígbà tí pẹpẹ bá da kọ́nù ní ọgbà sí ìlà lórí kọ́nù náà.
A hyperbola is a conic section formed when a plane intersects both halves of a double-napped cone.	Háípábólà jẹ́ abala kọ́nìkì tí a ṣẹ̀dá nígbà tí pẹpẹ bá da àwọn ààbọ̀ méjèèjì abala-méjì kọ́nù.
A differential equation is an equation involving derivatives of a function.	Ọ̀mì-ìyàtọ̀sọ́tọ̀ jẹ́ ọ̀mì tí ó ní àwọn dérífátífù ìfà.
A first-order differential equation involves only the first derivative of the function.	Ọ̀mì-ìyàtọ̀sọ́tọ̀ ìpele-àkọ́kọ́ ní dérífátífù àkọ́kọ́ ìfà náà nìkan.
A second-order differential equation involves the second derivative of the function.	Ọ̀mì-ìyàtọ̀sọ́tọ̀ ìpele-kejì ní dérífátífù kejì ìfà náà.
A linear differential equation has the dependent variable and all its derivatives occurring with power 1.	Ọ̀mì-ìyàtọ̀sọ́tọ̀ dogbá ní onídìmọ̀ aláṣeégbé àti gbogbo àwọn dérífátífù rẹ̀ tí ó farahàn pẹ̀lú agbára 1.
A partial differential equation involves partial derivatives of a function of several variables.	Ọ̀mì-ìyàtọ̀sọ́tọ̀ abẹ́lé ní àwọn dérífátífù abẹ́lé ìfà àwọn onídìmọ̀ púpọ̀.
A boundary value problem is a differential equation together with a set of boundary conditions.	Ìyọnu iye-ààlà jẹ́ ọ̀mì-ìyàtọ̀sọ́tọ̀ pẹ̀lú ìtò àwọn ipò ààlà.
An initial value problem is a differential equation together with a set of initial conditions.	Ìyọnu iye-ìbẹ̀rẹ̀ jẹ́ ọ̀mì-ìyàtọ̀sọ́tọ̀ pẹ̀lú ìtò àwọn ipò ìbẹ̀rẹ̀.
A mathematical model is a description of a system using mathematical concepts and language.	Àwòṣe ìṣirò jẹ́ àpèjúwe ètò nípa lílo àwọn èrò àti èdè ìṣirò.

Linear programming is a method to achieve the best outcome in a mathematical model with linear constraints.	Ìṣírò-dogbá jẹ́ ọ̀nà láti ṣe àṣeyọrí èsì tí ó dára jùlọ nínú àwòṣe ìṣirò pẹ̀lú àwọn ìdílọ́wọ́ dogbá.
Game theory is the study of mathematical models of strategic interaction between rational decision-makers.	Ẹ̀kọ́ ere jẹ́ ẹ̀kọ́ àwọn àwòṣe ìṣirò ìbáṣepọ̀ àdájọ láàrín àwọn olùpinnu aláròjinlẹ̀.
A zero-sum game is a game where one player's gain is exactly balanced by another player's loss.	Ere òdo-àpapọ̀ jẹ́ ere níbi tí èrè ẹlẹ́rẹ kan jẹ́ fífẹ̀gbálẹ̀ pẹ̀lú ikọlulẹ ẹlẹ́rẹ mìíràn.
A strategy is a complete plan of action in game theory.	Ìlànà-ẹ̀rí jẹ́ ètò pípé ìgbéṣẹ̀ nínú ẹ̀kọ́ ere.
A Nash equilibrium is a stable state in a game where no player can gain by unilaterally changing their strategy.	Ìdógba Nási jẹ́ ipò ìdúró nínú ere níbi tí kò sí ẹlẹ́rẹ tí ó lè jèrè nípa yíyi ìlànà-ẹ̀rí wọn padà lẹ́kansoso.
A graph in graph theory consists of vertices and edges.	Àwòrán nínú ẹ̀kọ́ àwòrán jẹ́ àkójọ àwọn tẹ́tẹ́ àti etí.
A directed graph has edges with a specific direction.	Àwòrán atọ́sọ́nà ní àwọn etí pẹ̀lú ìhà pàtó.
A weighted graph has a value associated with each edge.	Àwòrán alániwọ̀n ní iye tí a ṣe àbáṣepọ̀ pẹ̀lú etí kọ̀ọ̀kan.
A tree is a connected graph with no cycles.	Igi jẹ́ àwòrán asọpọ̀ pẹ̀lú kò sí àwọn àyídà.

A path in a graph is a sequence of vertices such that each adjacent pair is connected by an edge.	Ọ̀nà nínú àwòrán jẹ́ ìtẹ̀lé àwọn tẹ́tẹ́ tó bẹ́ẹ̀ tí ìwọ̀n tímọ́ kọ̀ọ̀kan jẹ́ sísopọ̀ nípa etí.
A cycle in a graph is a path that starts and ends at the same vertex.	Àyídà nínú àwòrán jẹ́ ọ̀nà tí ó bẹ̀rẹ̀ tí ó sì parí ní tẹ́tẹ́ kannáà.
